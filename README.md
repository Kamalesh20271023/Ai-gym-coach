# AI Gym Coach

AI-powered adaptive fitness coaching system built for AI Bharath Hackathon.

## Problem
Static gym plans lack adaptive progression and injury awareness.

## Solution
An AI-driven modular system that:
- Generates personalized workout plans
- Adapts progression
- Substitutes exercises during injury
- Tracks performance trends
